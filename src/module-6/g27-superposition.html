Superposition & Polysemanticity

Models represent far more concepts than dimensions via superposition — mathematically principled compression, not random noise.

~12 min
Anchor Analogy
A library fitting 100,000 books on 10,000 shelves by encoding each book as a combination of shelf positions.
Why This Matters
Answers "how can a finite model know so much?" — mathematical compression stores more knowledge than the architecture seemingly allows.
After This Page, You Can Argue
After this page, you can explain the compression trick: "The model stores more concepts than it has dimensions — using the same math that compresses music files."
