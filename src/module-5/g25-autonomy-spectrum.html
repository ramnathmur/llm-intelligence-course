The Autonomy Spectrum

Agent autonomy is a spectrum from fully human-controlled to fully autonomous, with meaningful engineering tradeoffs at each level.

~10 min
Anchor Analogy
Self-driving car levels (L0–L5): each trades control for independence, with real engineering consequences at each transition.
Why This Matters
This project is about autonomy. This gap gives the formal vocabulary for choosing where each agent should operate on the spectrum.
After This Page, You Can Argue
After this page, you can make informed decisions about how much autonomy to give each agent — and explain why "more autonomy" isn't always better.
