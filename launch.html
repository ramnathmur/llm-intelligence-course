Why This Course Matters

If you build with or study LLMs, the "next-word guesser" frame is dangerously incomplete. It explains away genuine intelligence as statistical mimicry—but the evidence is overwhelming: LLMs exhibit reasoning, transfer learning, tool use, and goal-directed behavior that go far beyond pattern matching.

This course answers the central question: If an LLM just predicts the next word—where does genuine intelligence come from? By the end, you'll have a complete, defensible understanding of how intelligence emerges, what makes it real, and how it becomes autonomous action.

Getting Started (Quick Setup)

Step 1: Navigate to the Course Directory

Open a terminal or PowerShell window and navigate to the course folder:

cd "C:\Claude Cowork\Projects\ai autonomy"

Step 2: Start a Local Web Server

Run one of the commands below (choose the one that matches your system):

Python 3 (Recommended):

python -m http.server 8000

Python 2 (if Python 3 isn't installed):

python -m SimpleHTTPServer 8000

Node.js (if you have it installed):

npx http-server -p 8000
✓ You should see a message like "Serving HTTP on port 8000". Leave this terminal open while you read the course.

Step 3: Open the Course in Your Browser

Once the server is running, open this link in your browser:

http://localhost:8000/launch.html

Then click the "Launch Course" button below to enter the full curriculum.

Why Do We Need a Server?

Browsers block certain JavaScript functionality when pages are opened directly as files (the file:// protocol). The course uses localStorage to track your progress, which requires a web server. Starting a local server (like the one above) solves this without putting anything on the internet—it all runs on your computer.
